## Tech Stacks:
- Frontend: ReactJS
- Backend: ExpressJS
- Database: MongoDB
- UI Design: Bootstrap

## Install the dependencies
```bash
npm i
```

## Run the development server
```bash
npm run dev
```

## Run the production server
```bash
npm run start
```
